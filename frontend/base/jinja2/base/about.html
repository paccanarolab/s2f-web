Welcome to S2Fweb

Protein Function Prediction for newly sequenced organisms

S2Fweb is a web server that allows users to predict protein functions for newly sequenced organisms using the S2F (Sequence to Function) method.

Developed by Mateo Torres and Marcelo Baez
